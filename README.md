# 上海吉斯普机械设备有限公司官网

## 网站概述
上海吉斯普机械设备有限公司（GISPERT Mining Machinery Co., Ltd.）官方网站，展示公司产品、服务、代理品牌和联系信息。

## 网站结构
- `index.html` - 首页
- `about.html` - 关于我们
- `products.html` - 产品中心
- `brands.html` - 代理品牌
- `services.html` - 技术服务
- `contact.html` - 联系我们

## 技术栈
- HTML5
- CSS3 (自定义样式)
- JavaScript (基础交互)
- 响应式设计

## 功能特点
1. 响应式设计，适配各种设备
2. 清晰的导航结构
3. 产品分类展示
4. 品牌合作伙伴介绍
5. 技术服务流程展示
6. 联系表单和地图集成

## 部署说明
本网站通过GitHub Pages部署，支持自定义域名。
